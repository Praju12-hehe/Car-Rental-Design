Project Name: Varda Car Rental

Design Reference: Elyse Residence Luxury Real Estate (High-end typography, fluid scroll animations, expansive whitespace, and premium image reveals).

Core Objective: Build a luxury car rental web application with a sophisticated animation engine and a deep-link WhatsApp booking system.

1. Technical Stack & Visual Style:

Tech: HTML5, Tailwind CSS, Framer Motion, and Lucide Icons.

Aesthetic: Follow the "Elyse Residence" Dribbble reference: use elegant serif headings, smooth parallax scrolling, and refined hover states.

Glassmorphism: Use semi-transparent, blurred backgrounds for UI cards and the sticky navigation bar.

2. Dynamic Features:

Product Detail View (Flipkart-style): When a user clicks a car card, do not navigate to a new page. Instead, open a sleek, full-screen overlay or a "drawer" containing:

High-resolution car image gallery.

Specifications: Seating capacity, fuel type, and transmission.

Detailed pricing (Manual vs. Auto).

A persistent "Book via WhatsApp" button specific to that car.

Advanced Booking Window: A dedicated booking modal featuring:

Input fields for Name, Pickup Place, and Drop Place.

An interactive Calendar Picker for dates and a Time Picker.

A "Confirm & Send to WhatsApp" button that compiles all these inputs into a pre-filled WhatsApp message to 7666357013.
